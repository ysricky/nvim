# nvim
My personal Neovim config files

## Requirements:
- [Neovim 0.7.2](https://github.com/neovim/neovim/releases/)
- clipboard (xclip or wl-clipboard)

## Installation:
- Install [Packer](https://github.com/wbthomason/packer.nvim)
- Run this command:
```shell
nvim +PackerSync
```
